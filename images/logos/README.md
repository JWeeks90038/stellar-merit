# Logo File

Place your logo file in this directory:

**File name:** `StellarMerit Statuary Logo_Lion.png`

The logo will automatically appear in the navigation bar on all pages.

## Logo Specifications

- **Current setup:** PNG format
- **Recommended size:** 200-400px width (height will auto-adjust)
- **Background:** Transparent PNG works best
- **File name:** Must be exactly `StellarMerit Statuary Logo_Lion.png`

## How to Add Your Logo

1. Copy `StellarMerit Statuary Logo_Lion.png` to this directory (`E:\stellar-merit\images\logos\`)
2. Refresh your browser to see the logo in the navigation bar
3. The logo will appear on all pages (Home, Shop, About, Contact)

## Logo Display

- **Desktop:** Logo displays at 50px height with text beside it
- **Tablet:** Logo displays at 40px height
- **Mobile:** Logo displays at 35px height

The logo scales responsively and maintains its aspect ratio on all devices.
